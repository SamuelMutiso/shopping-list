# Interactive Shopping List 
A simple, dynamic web application built with HTML, CSS, and JavaScript that allows users to manage their shopping needs in real-time.

##  Features
Add Items:-Quickly add new items to your list.
ark as Purchased:- Toggle the status of items with a single click.
Clear List:- Remove all items to start fresh.
Persistent Data:-Uses local storage to save your list even after refreshing the page.

## Technologies Used
HTML5 - Structure
CSS3 - Styling & Layout
JavaScript- Logic & DOM Manipulation
